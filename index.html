        How To Use Study Arcade

        Quick Start

        1. Click Load Terms .txt and choose a compatible terms file (for example, terms.txt).
        2. Check that the source line says a file is loaded.
        3. Select one or more topics.
        4. Choose game type and speed (or Mission Accessible options when that mode is selected).
        5. Click Start Mission.

        Terms File

        Use the provided terms.txt template. Keep the instruction header at the top, including the line Study Arcade, copyright Dr. Grant Benham. Do not delete that line.

        Game Modes

        Term Invaders: move left/right to choose a lane, then fire at the term that matches the current definition.

        Banner Drive: steer left/right and pass under the banner that matches the current definition.

        Signal Lock: choose a signal lane and lock onto the matching term for the current definition.

        Mission Accessible: fully DOM-based gameplay with three large answer choices per prompt, optional timing, and keyboard-first input.

        Controls

        Term Invaders/Banner Drive/Signal Lock: Left/Right controls or arrow keys, P pause/resume, K skip. Fire controls apply to Term Invaders and Signal Lock.

        Mission Accessible: use 1, 2, 3 to select choices, Enter or Space to submit when confirm mode is enabled, H for hint, R to repeat the definition, plus P and K. Mission pause/skip buttons are also in the Game Arena for easier access while focus is locked.

        Mission Accessible keeps keyboard focus cycling inside the mission arena while active. Use Adjust Settings to temporarily unlock focus and jump to Setup controls, then use Return to Mission Focus to lock focus back to gameplay.

        If you change settings during a mission, the game auto-pauses. Changing speed resets the current wave/set in Term Invaders and Banner Drive, while Mission Accessible applies updated options to the next prompt.

        Use Sound Level to set audio to Off, Low, Medium, or High (default Medium).

        Scoring

        All game types use three choices per prompt. Correct answers add points and wrong answers reduce points. In Term Invaders, city integrity drops when terms reach the city. Wrong Answer Limit (Off/5/10/20) applies to Term Invaders and Banner Drive.

        The Missed Terms panel updates live during missions and shows missed/incorrect items.
