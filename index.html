      How To Use Study Arcade

      Quick Start

      1. Click Load Terms .txt and choose a compatible terms file (for example, terms.txt).
      2. Check that the source line says a file is loaded.
      3. Select one or more topics.
      4. Choose game type, choices per round, and speed.
      5. Click Start Mission.

      Terms File Format

      Use topic headers and term-definition lines:

      # Topic Name
Term: Definition
Another term: Another definition

      Game Modes

      Term Invaders: move left/right to choose a lane, then fire at the term that matches the current definition.

      Banner Drive: steer left/right and pass under the banner that matches the current definition.

      Controls

      Buttons: Left, Right, and Fire (Term Invaders only). Keyboard: arrow keys to move, Enter/Space to fire in Term Invaders, P to pause/resume, and K to skip.

      Scoring

      Correct answers add points. Wrong answers reduce points. In Term Invaders, city integrity drops when terms reach the city.
